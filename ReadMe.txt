Steps to run the Web application


1) Make sure you have python 3's any stable version installed on your PC

2) open the command prompt in the location where this folder is saved

to be more precise open the command line prompt in the directory where the python fine "app1.py is located"

3) type the command " python app1.py " in the command prompt or the terminal

4) It will give a http url to copy on your local browser like this " http://127.0.0.1:5000/ "

5) Once you have copied the url on the browser press enter that will bring you to the User interface

6)Put in appropriate data as mentioned in the label

7) With appropriate information given the Machine Learning model will give out a prediction

8)You can enter new values again to test out the algorithm and to make loan default prediction.
